# Лабораторная работа №2 - Разработка многостраничного веб-сайта
## Цель работы
Изучение основ HTML и CSS, создание многостраничного веб-сайта с соблюдением требований к разметке и стилям.

## Тема проекта
Веб-сайт "Космос и Астрономия" - образовательный ресурс о планетах Солнечной системы и космических явлениях.

## Структура проекта
- `index.html` - главная страница
- `planets.html` - страница с таблицей планет
- `gallery.html` - галерея космических изображений
- `contact.html` - страница контактов
- `styles.css` - файл стилей
- `images/` - папка с изображениями

## Реализованные требования

### HTML
- ✅ 4 страницы
- ✅ Блочные и строчные элементы
- ✅ Ссылки (навигация между страницами)
- ✅ Картинки (в галерее)
- ✅ Таблица (с информацией о планетах)

### CSS
- ✅ Отдельный подключаемый файл
- ✅ Использовано 3+ видов селекторов:
  - Селекторы по тегу (`body`, `table`, `h1`)
  - Селекторы по классу (`.header`, `.gallery-item`)
  - Селекторы по ID (`#main-title`)
  - Комбинированные селекторы (`table th, table td`)
- ✅ Использование цвета
- ✅ Разные виды выравнивания в таблице

## Особенности реализации
- Адаптивный дизайн
- Семантическая разметка
- Современные CSS-свойства
- Чистая и понятная структура кода

### Как запустить
Открой lab 2/index.html в браузере.

- Адаптивный дизайн (медиа-запросы)
- БЭМ-методология именования классов

## Лабораторная работа 5

### Цель
Изучение современного CSS, обработки событий на JavaScript и асинхронности.

### Реализация
1. **Динамическая проверка данных формы** в процессе ввода с использованием событийной модели JavaScript
2. **Отправка данных формы** после валидации через POST-запрос (jsonplaceholder.typicode.com)
3. **Асинхронный запрос на сервер** для получения данных о планетах
4. **Периодическое обновление данных** (каждые 5 минут) с обработкой ошибок

### Используемые технологии
- HTML5, CSS3 (Flexbox, Grid, CSS-переменные)
- JavaScript (ES6+, Fetch API, async/await, обработка событий)
- JSON Server (mock-сервер для данных)

### Структура
- `lab5/form.html` - форма с динамической валидацией
- `lab5/planets.html` - таблица с асинхронной загрузкой данных
- `lab5/db.json` - данные для mock-сервера
- `lab5/script.js` - логика валидации и асинхронных запросов
- `lab5/styles.css` - стили для подсказок валидации
