# Лабораторная работа №2 - Разработка многостраничного веб-сайта
## Цель работы
Изучение основ HTML и CSS, создание многостраничного веб-сайта с соблюдением требований к разметке и стилям.

## Тема проекта
Веб-сайт "Космос и Астрономия" - образовательный ресурс о планетах Солнечной системы и космических явлениях.

## Структура проекта
- `index.html` - главная страница
- `planets.html` - страница с таблицей планет
- `gallery.html` - галерея космических изображений
- `contact.html` - страница контактов
- `styles.css` - файл стилей
- `images/` - папка с изображениями

## Реализованные требования

### HTML
- ✅ 4 страницы
- ✅ Блочные и строчные элементы
- ✅ Ссылки (навигация между страницами)
- ✅ Картинки (в галерее)
- ✅ Таблица (с информацией о планетах)

### CSS
- ✅ Отдельный подключаемый файл
- ✅ Использовано 3+ видов селекторов:
  - Селекторы по тегу (`body`, `table`, `h1`)
  - Селекторы по классу (`.header`, `.gallery-item`)
  - Селекторы по ID (`#main-title`)
  - Комбинированные селекторы (`table th, table td`)
- ✅ Использование цвета
- ✅ Разные виды выравнивания в таблице

## Особенности реализации
- Адаптивный дизайн
- Семантическая разметка
- Современные CSS-свойства
- Чистая и понятная структура кода

### Как запустить
Открой lab 2/index.html в браузере.



# Лабораторная работа №3 — Основы JavaScript
## Цель работы
Освоение базовых концепций программирования на JavaScript, включая работу с функциями, рекурсией, замыканиями и объектами.

## Описание выполненной работы
В ходе лабораторной работы были разработаны различные функции на JavaScript, демонстрирующие ключевые аспекты языка:

## Реализованные функции и их назначение:
Проверка целых чисел - функция для определения целочисленности значения

Генератор чётных чисел - создание последовательности чётных чисел в заданном диапазоне

Вычисление суммы чисел - две реализации (итеративная и рекурсивная) для накопления суммы

Факториал числа - вычисление произведения чисел от 1 до n

Проверка степени двойки - определение, является ли число степенью числа 2

Числа Фибоначчи - нахождение элементов знаменитой числовой последовательности

Функции-замыкания - создание функций с сохранением состояния

Генератор последовательностей - генерация арифметических прогрессий

Глубокое сравнение объектов - рекурсивная проверка эквивалентности объектов

## Особенности реализации
Использованы различные подходы: итеративные и рекурсивные решения

Применены побитовые операции для оптимизации проверок
